# vfox-stacker

[![CI](https://github.com/jbox-web/vfox-stacker/workflows/CI/badge.svg)](https://github.com/jbox-web/vfox-stacker/actions)

A plugin for the [mise version manager](https://github.com/jdx/mise) to install [Stacker](https://github.com/jbox-web/stacker)

## Install

```
mise plugin install stacker https://github.com/jbox-web/vfox-stacker.git
```
